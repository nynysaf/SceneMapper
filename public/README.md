# Static assets

Place the **Scene Mapper logo** here so it appears in the header (landing and dashboard).

- **Filename:** `logo.png` (or `logo.svg`)
- **Recommended dimensions:** **64×64 px** or **96×96 px** (for sharp display on retina)
- **Format:** PNG with transparency, or SVG

The header uses a 40×40 px display size, so 64×64 or 96×96 will look crisp. Drop your file as:

```
public/logo.png
```

The app will use it at `/logo.png`. If the file is missing, the header shows the "SM" text fallback.
